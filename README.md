Project Overview

This is a full-stack Task Assignment System built using the MERN stack.
It allows admins to assign tasks and users to manage and track their work efficiently.



Frontend: React.js, Material UI
Backend: Node.js, Express.js
Database: MongoDB
Version Control: Git & GitHub


🔐 User Authentication (Login/Register)
👨‍💼 Admin Dashboard
📋 Task Assignment to Users
📊 Task Status Tracking
🧾 API Integration (Frontend ↔ Backend)
📱 Responsive UI
